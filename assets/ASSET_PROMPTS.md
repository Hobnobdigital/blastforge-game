## Generation Script

Save this as `generate_assets.py`:

```python
import requests
import json
import base64
import os

api_key = os.environ.get('OPENAI_API_KEY')
if not api_key:
    print("Please set OPENAI_API_KEY environment variable")
    exit(1)

# Load prompts from this document
# Generate each asset with:
# - Model: gpt-image-1 or gpt-image-1.5
# - Size: 1024x1024 (or 1536x1024 for landscapes)
# - Quality: high
# - Format: png

# Example generation function:
def generate_image(prompt, filename, size="1024x1024"):
    response = requests.post(
        "https://api.openai.com/v1/images/generations",
        headers={"Authorization": f"Bearer {api_key}"},
        json={
            "model": "gpt-image-1",
            "prompt": prompt,
            "n": 1,
            "size": size,
            "quality": "high"
        }
    )
    data = response.json()
    if 'data' in data:
        image_data = base64.b64decode(data['data'][0]['b64_json'])
        with open(filename, "wb") as f:
            f.write(image_data)
        print(f"Generated: {filename}")
    else:
        print(f"Error: {data}")
```
