# Vendor Directory

This file contains configuration for known vendors including contact info, pricing structure, and website details.

## Creative Boulevard

**Website:** https://www.creativeboulevard.com  
**Type:** Quote-based (custom branded merchandise)  
**Specialties:** Promotional products, swag items, custom branding  
**Contact Method:** Email quote request  
**Typical Lead Time:** 2-3 weeks for custom items  

**Quote Request Info:**
- Always mention BCEHS branding requirement
- Specify quantities needed
- Request tiered pricing (multiple quantity levels)
- Ask about shipping to BC location
- Request setup/design fees separately

## Precision Graphics

**Website:** https://www.precisiongraphics.ca  
**Type:** Quote-based (commercial printing)  
**Location:** Vancouver, BC  
**Specialties:** Printing, copying, prepress, desktop design  
**Contact Method:** Email or phone consultation  
**Typical Lead Time:** Varies by project  

**Quote Request Info:**
- Mention custom BCEHS branding/logo
- Provide quantity ranges
- Ask about bulk discounts
- Shipping usually local (Vancouver-based)

## SwagZone

**Website:** https://cat-ca.swag-zone.com  
**Type:** Interactive web pricing (slider-based)  
**Specialties:** Custom merchandise, promotional products, branded swag  
**Location:** Nationwide delivery across Canada  
**Pricing Method:** Online catalog with quantity sliders  

**Automation Notes:**
- Uses quantity slider/range input for pricing
- Prices update dynamically as slider moves
- May require account for exact pricing on some items
- See `references/browser-automation.md` for slider interaction patterns

**Quote Request Info:**
- Can also request custom quotes via contact form
- Mention BCEHS branding requirements
- Typical turnaround: 2 weeks
- High-quality products (not cheap junk)
