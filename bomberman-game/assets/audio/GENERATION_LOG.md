## ‚ö†Ô∏è Important Notes

### Reproducibility
All AI-generated assets should be reproducible using the prompts and settings logged above. If regeneration is needed:
1. Use the exact prompt
2. Apply the same settings
3. Select similar variations
4. Apply same post-processing

### Version Control
- Keep original files (before processing) in `temp/originals/`
- Don't delete until project is complete
- Helps if regeneration is needed

### License Updates
AI service terms change frequently. Always:
- Check current terms before commercial release
- Save a copy of terms at time of generation
- Update this log if terms change

### Attribution
If any assets require attribution:
1. Add to CREDITS.md in project root
2. Include in game credits screen
3. Format: "[Asset Name] by [Creator/Service] ([License])"
